GET /get_matches_result/{k}

Пути счёта матча

Рекурсивно перебирает все возможные пути, которыми счёт матча может дойти от 0:0 до k:k. Каждый путь — это последовательность промежуточных счётов.

GET /use_memory_decorator/{index_func}

Замер памяти функции

Выполняет выбранную функцию и измеряет потребление памяти с помощью tracemalloc.

1
list_of_squares
Список квадратов чисел от 0 до n−1
count: int — количество элементов
2
generator_of_squares
То же, но через генераторное выражение (list comprehension через generator)
count: int — количество элементов
3
fibonacci_recursive
n-е число Фибоначчи через рекурсию. Память растёт из-за стека вызовов. Не вводите n > 20!
count: int — индекс числа Фибоначчи
4
matrix_multiply
Перемножение двух матриц size×size через вложенные списки
count: int — размер матрицы
5
word_frequency
Подсчёт частоты каждого слова в тексте. Возвращает словарь word → count
text: str — произвольный текст для анализа
