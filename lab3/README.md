### Задача 1

Используя `itertools.permutations`, генерируем все перестановки из 5 различных цифр восьмеричной системы (0–7). Отсекаем числа, начинающиеся с нуля. Для каждого кода проверяем, что ни две чётные ({0, 2, 4, 6}), ни две нечётные ({1, 3, 5, 7}) цифры не стоят рядом.

```python
from itertools import permutations

def solve_ex1():
    digits = [0, 1, 2, 3, 4, 5, 6, 7]
    even_digits = {0, 2, 4, 6}
    odd_digits  = {1, 3, 5, 7}
    count = 0
    for code in permutations(digits, 5):
        if code[0] == 0:
            continue
        flag = True
        for i in range(len(code) - 1):
            digit, next_d = code[i], code[i + 1]
            if digit in even_digits and next_d in even_digits:
                flag = False
                break
            if digit in odd_digits and next_d in odd_digits:
                flag = False
                break
        if flag:
            count += 1
    return count
```

### Задача 2

Вычисляем значение выражения средствами Python (BigInteger), затем переводим его в девятичную систему счисления последовательным делением на 9. Подсчитываем количество вхождений цифры `8` в полученной строке.

```python
def solve_ex2():
    expression = (81**17) + (3**24) - 45

    def translator(num):
        res = ''
        while num > 0:
            res = str(num % 9) + res
            num //= 9
        return res

    new_expr = translator(expression)
    return new_expr.count("8")
```

### Задача 3

Числа с ровно тремя нетривиальными делителями имеют вид $p^4$ (простое $p$), так как делители $p^4$ — это $1, p, p^2, p^3, p^4$, из которых нетривиальные: $p, p^2, p^3$. Перебираем простые $p$ в диапазоне $[\sqrt[4]{123456789},\, \sqrt[4]{223456789}]$, проверяем попадание $p^4$ в отрезок и считаем нетривиальные делители. Выводим само число и его наибольший нетривиальный делитель.

```python
def solve_ex3():
    def find_divs(number):
        divs = set()
        for i in range(2, int(number ** 0.5) + 1):
            if number % i == 0:
                divs.add(i)
                divs.add(number // i)
        return divs

    for number in range(int(123456789 ** 0.25), int(223456789 ** 0.25) + 1):
        if 123456789 <= number ** 4 <= 223456789:
            divs = find_divs(number ** 4)
            if len(divs) == 3:
                print(number ** 4, max(divs))
```
