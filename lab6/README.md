## Условие задачи

### Создать генератор цифр числа π

Реализовать генератор, генератор цифр числа π

## Описание проделанной работы

```python
def pi_leibniz(iterations: int) -> float:
    pi = 0.0
    for k in range(iterations):
        pi += ((-1) ** k) / (2 * k + 1)
        yield pi * 4
```
Механизм работы:
- Функция реализует формулу Лейбница для вычисления π: π/4 = 1-1/3 + 1/5 - 1/7 + ...
- На каждой итерации k прибавляет к сумме член ((-1) ** k) / (2 * k + 1) -- знак чередуется через (-1) ** k, знаменатель растет на 2.
- После каждого шага yield-ит текущее приближение pi * 4, то есть это генератор -- он возвращает промежуточные значения π по одному на каждой итерации, постепенно уточняя результат.
